# Network Reconnaissance and Enumeration Unified Toolkit

This toolkit provides a single command-line interface to execute multiple
Kali Linux reconnaissance and enumeration tools.

##Usage:
python3 recon_tool.py <tool> <target>


## Supported Tools
- nmap
- nikto
- whatweb
- netdiscover
- arp-scan
- ping
- traceroute
- nslookup
- dig
- nbtscan
- smb

## Example
./recon-tool nmap 192.168.1.10
./recon-tool nikto http://example.com

This tool is intended for educational and authorized testing only.
